Scenario 3: No Assets Found üö´

What You'll See:

[INSERT SCREENSHOT]
Error view showing: - Orange warning triangle icon - Error message: "We couldn't find the datasets powering this report in Acceldata" - "Fetch Again" button - "Logout" button

What This Means

This error appears when:

  • The Power BI report uses datasets that aren't tracked in ADOC yet
  • Dataset names don't match between Power BI and ADOC
  • You don't have permission to view these assets in ADOC
  • There's a temporary network issue

Important: This doesn't necessarily mean your data is bad! It just means ADOC doesn't have information about these specific datasets.

What You Should Do

If This Happens... Do This...
First time seeing this error
  1. Click "Fetch Again" to retry
  2. If still fails, check your internet connection
  3. Verify you can access ADOC platform directly
Error persists
  1. Contact your ADOC administrator
  2. Request onboarding for these datasets
  3. Verify you have permissions in ADOC
Need to switch accounts
  1. Click "Logout" button
  2. Login with different credentials
  3. Try fetching again
Want to test with different report
  1. Open a different Power BI report
  2. Click ADOC icon
  3. See if that report's data is tracked

üí° Workaround:

If your datasets aren't in ADOC yet, use traditional data quality checks until onboarding is complete. Meanwhile, bookmark reports that DO work with ADOC as your "verified" reports.

Troubleshooting

Problem: Extension icon is grayed out

[INSERT SCREENSHOT]
Chrome toolbar showing grayed out ADOC icon (inactive state)

Solution:

  1. Navigate to a Power BI report page (app.powerbi.com)
  2. Icon should turn active/colored
  3. If still gray, refresh the page

üí° Why This Happens:

Extension only activates on Power BI pages for security. On other websites, the icon appears gray and inactive.

Problem: Stuck on "Fetching..." screen

[INSERT SCREENSHOT]
Fetching screen stuck with spinner animation

Solution:

  1. Wait 10 seconds (might be slow network)
  2. If still stuck: Close popup and click icon again
  3. If persists: Check internet connection
  4. Last resort: Logout and login again

Problem: "No Assets Found" for all reports

Possible Causes:

  • Datasets not onboarded to ADOC platform
  • Permission issues in ADOC
  • Network/firewall blocking ADOC API

Solution:

  1. Test with a known-good report (ask colleagues)
  2. Login to ADOC platform directly and verify access
  3. Contact ADOC administrator: support@acceldata.io

Problem: Login doesn't redirect back

Solution:

  1. After login, manually close the auth tab
  2. Wait for success notification
  3. Click extension icon again
  4. Extension should show fetching/results
